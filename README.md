# 📚 Library Management System

A console-based **Library Management System** developed in **Java** using Object-Oriented Programming (OOP). This application allows users to manage books efficiently with features like adding, updating, deleting, issuing, returning, and searching books. Data is also stored using file handling to preserve records between program runs.

---

## 🚀 Features

- ➕ Add New Book
- 📖 Display All Books
- 🔍 Search Book by ID
- 🔎 Search Book by Title
- 👨‍💼 Search Book by Author
- 📂 Search Book by Category
- ✏️ Update Book Details
- ❌ Delete Book
- 📤 Issue Book
- 📥 Return Book
- 📊 Library Statistics
- 📁 File Handling (Serialization)
- ✅ Input Validation
- 🧾 Menu-Driven Console Interface
